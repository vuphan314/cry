# Cry
cryptographic framework

------------------------------------------------------------
## prerequisites

### GMP
- [Ubuntu][gmp]
- macOS
  ```
  /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  brew install gmp
  ```

### cloning
`git clone --recursive https://github.com/vuphan314/cry`
